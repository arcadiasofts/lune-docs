## 📢 Orbital 베타 버전 출시

안녕하세요, Arcadia 팀입니다.

드디어 **Orbital 프레임워크 베타 버전**을 출시하게 되었습니다! 🎉

### 🚀 주요 기능

- **LUNE 프로토콜**: 고성능 커스텀 네트워크 프로토콜
- **Express.js 스타일 API**: 친숙한 라우팅과 미들웨어 시스템
- **이벤트 기반 아키텍처**: 실시간 양방향 통신 지원
- **강력한 타입 시스템**: Rust의 메모리 안정성과 성능

### 📖 시작하기

1. [Getting Started 가이드](/docs/orbital/getting-started)를 확인해보세요
2. [예제 코드](https://github.com/arcadiasofts/orbital-examples)를 실행해보세요
3. [API 문서](/docs/orbital/application)에서 자세한 사용법을 확인하세요

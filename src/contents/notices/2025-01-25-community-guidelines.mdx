## 🤝 커뮤니티 가이드라인

Orbital 프로젝트에 관심을 가져주셔서 감사합니다!

건전하고 생산적인 커뮤니티를 만들기 위한 가이드라인을 안내드립니다.

### 📋 기여 방법

#### 1. 코드 기여

```bash
# 1. 저장소 포크
git clone https://github.com/yourusername/orbital.git

# 2. 새 브랜치 생성
git checkout -b feature/awesome-feature

# 3. 변경사항 커밋
git commit -m "feat: Add awesome feature"

# 4. Pull Request 생성
```

#### 2. 문서 개선

- 오타 수정, 번역, 예제 추가
- [문서 저장소](https://github.com/arcadiasofts/orbital-docs)
- 마크다운 형식으로 작성

#### 3. 이슈 리포트

```markdown
## 버그 리포트 템플릿

### 환경

- OS: Ubuntu 22.04
- Rust 버전: 1.75.0
- Orbital 버전: 0.1.0-beta

### 재현 단계

1. 서버 시작
2. /api/test 요청
3. 에러 발생

### 예상 결과

정상 응답

### 실제 결과

500 Internal Server Error

### 추가 정보

로그 파일, 스크린샷 등
```

### 🎯 우선순위 영역

현재 다음 영역에서 기여를 특히 환영합니다:

#### 높은 우선순위

- **성능 최적화**: 벤치마크 및 프로파일링
- **에러 처리**: 더 나은 에러 메시지와 처리
- **테스트**: 단위 테스트 및 통합 테스트
- **문서**: 예제와 가이드 추가

#### 보통 우선순위

- **미들웨어**: 새로운 미들웨어 개발
- **유틸리티**: 개발자 도구 및 헬퍼
- **예제**: 실제 사용 사례 예제
- **번역**: 다국어 문서 지원

### 🏆 기여자 인정

기여해주신 모든 분들을 다음과 같이 인정합니다:

#### GitHub 기여자

- README의 Contributors 섹션에 자동 추가
- 릴리즈 노트에 기여 내용 명시

#### 특별 기여자

- **Core Contributors**: 지속적인 코드 기여자
- **Documentation Champions**: 문서 개선 기여자
- **Community Leaders**: 커뮤니티 활동 기여자

### 📞 소통 채널

#### 개발 관련

- **GitHub Discussions**: 기능 제안, 질문
- **GitHub Issues**: 버그 리포트, 작업 추적
- **Pull Requests**: 코드 리뷰

#### 커뮤니티

- **Discord**: [Arcadia 서버](https://discord.gg/arcadia)
  - `#orbital-general`: 일반 토론
  - `#orbital-dev`: 개발 관련 논의
  - `#orbital-help`: 도움 요청

#### 공식 소통

- **이메일**: community@arcadiasofts.com
- **블로그**: 월간 업데이트 및 소식

### 🚫 금지 사항

다음 행위는 금지됩니다:

- **스팸**: 반복적이고 무의미한 게시물
- **욕설**: 공격적이거나 모욕적인 언어
- **광고**: 관련 없는 제품이나 서비스 홍보
- **저작권 침해**: 허가 없는 코드 복사

### 🎉 커뮤니티 이벤트

#### 정기 이벤트

- **월간 개발자 미팅**: 매월 첫째 주 토요일
- **코딩 챌린지**: 분기별 개최
- **해커톤**: 연 2회 개최

#### 특별 이벤트

- **베타 테스터 모집**: 새 기능 테스트
- **문서 번역 마라톤**: 다국어 지원 확대
- **성능 최적화 컨테스트**: 벤치마크 개선

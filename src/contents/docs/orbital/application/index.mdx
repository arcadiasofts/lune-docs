## Orbital Application

`OrbitApplication`은 Orbital 프레임워크의 핵심 컴포넌트로, LUNE 프로토콜 기반의 서버 애플리케이션을 구성하고 관리합니다. Express.js와 유사한 API를 제공하면서도 Rust의 성능과 안전성을 활용할 수 있습니다.

## 주요 기능

- **Express 스타일 API**: 친숙한 웹 프레임워크 패턴
- **LUNE 프로토콜 지원**: 커스텀 네트워크 프로토콜
- **이벤트 기반 아키텍처**: 강력한 이벤트 시스템
- **실시간 모니터링**: 애플리케이션 상태 추적
- **Graceful Shutdown**: 안전한 서버 종료
- **설정 관리**: 유연한 구성 옵션

## 빠른 시작

```rust
use orbital::application::OrbitApplication;
use orbital::router::{Router, RouteResponse};
use serde_json::json;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut app = OrbitApplication::new(
        Some("My LUNE Server".to_string()),
        Some("LUNE 프로토콜 기반 서버".to_string()),
        Some("1.0.0".to_string())
    );

    let mut router = Router::new("Main Router");
    router.read("/", Arc::new(|_ctx| {
        Ok(RouteResponse::json(json!({
            "message": "Hello, Orbital!",
            "protocol": "LUNE/1"
        })))
    }));

    app.register(router)?;
    app.listen(8080).await?;

    Ok(())
}
```

## API 레퍼런스

### OrbitApplication 메서드

| 메서드                         | 설명                          |
| ------------------------------ | ----------------------------- |
| `new(name, desc, version)`     | 새로운 애플리케이션 생성      |
| `with_max_connections(...)`    | 최대 연결 수 제한과 함께 생성 |
| `register(router)`             | 단일 라우터 등록              |
| `register_routers(routers)`    | 여러 라우터 한번에 등록       |
| `listen(port)`                 | 지정된 포트에서 서버 시작     |
| `stop()`                       | 서버 중지 및 연결 종료        |
| `status()`                     | 애플리케이션 상태 조회        |
| `is_running()`                 | 실행 상태 확인                |
| `router_count()`               | 등록된 라우터 수 반환         |
| `router_statistics()`          | 라우터 통계 조회              |
| `connection_manager()`         | 연결 관리자 반환              |
| `event_emitter()`              | 이벤트 에미터 반환            |
| `emit_event(name, payload)`    | 커스텀 이벤트 발생            |
| `log(level, message, context)` | 구조화된 로그 출력            |

### ApplicationStatus 필드

| 필드                  | 타입                   | 설명              |
| --------------------- | ---------------------- | ----------------- |
| `name`                | `String`               | 애플리케이션 이름 |
| `version`             | `String`               | 애플리케이션 버전 |
| `is_running`          | `bool`                 | 실행 상태         |
| `connection_stats`    | `ConnectionStatistics` | 연결 통계         |
| `active_connections`  | `usize`                | 활성 연결 수      |
| `healthy_connections` | `usize`                | 정상 연결 수      |
